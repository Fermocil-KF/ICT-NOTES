Data Types in Python

In Python, all data is represented by objects, each having an Identity (ID), Type, and Value.

  • Integer (int): Whole numbers (e.g., 10, -5).
  • Floating Point (float): Numbers with decimals (e.g., 3.14, -0.5).
  • Boolean (bool): Either True or False.
  • String (str): Sequence of characters (e.g., "hello").
  • List (list): Ordered, mutable collection of items (e.g., [1, 'a']).
  • Tuple (tuple): Ordered, immutable sequence of items (e.g., (4, 18)).
  • Dictionary (dict): Ordered collection of key/value pairs (e.g., {'key': 'value'}).

Mutability: Refers to whether an object's value can be changed after creation (e.g., Lists are mutable; Tuples are immutable).

Working with Numbers & Operators

Python follows PEMDAS for order of operations.

Addition:        +
Subtraction:     -
Multiplication:  *
Exponentiation:  **
Division (float): /  (e.g., 5 / 2 = 2.5)
Floor Division:  // (e.g., 5 // 2 = 2)
Modulo/Remainder: %  (e.g., 5 % 2 = 1)

Type Conversion: Use int(), float(), or bool() to convert between types. Operations between a float and an int result in a float.

Strings (str)

Strings are composed of a sequence of characters. They are specified by enclosing characters in single (') or double (") quotation marks. You can also specify multi-line strings using three quotation marks (e.g., """...""").

String Operations and Indexing

  • Concatenation & Multiplication: Use the `+` operator to join strings together. You can multiply a string using the `*` operator to repeat it (e.g., `"Hi" * 3` gives `"HiHiHi"`).
  • Offset/Indexing: The position of a character is called its offset. Python uses **zero-based indexing** (starts counting from 0).
    • Positive Indexing: Counts from left to right (0, 1, 2, ...).
    • Negative Indexing: Counts from right to left, where `-1` is the last character.
  • Length: The built-in `len()` function returns the number of characters in the string as an integer.

Escape Characters

Escape characters, signaled by a backslash (`\`), instruct Python to treat the following character specially, not as a literal symbol.

String Methods

  • Strip Method: Removes unwanted characters (by default, whitespace) from the start and end of a string.
    • `.strip()`: Strips whitespace on both sides.
    • `.lstrip()`: Strips whitespace on the left side.
    • `.rstrip()`: Strips whitespace on the right side.
  • Split and Join Methods:
    • `.split()`: Converts a string into a **list** of substrings based on a delimiter (defaults to whitespace).
    • `.join()`: Combines the items of a list back into a single string, using the string on which the method is called as the separator.
  • Replace Method: The `.replace(old, new)` method replaces all occurrences of the `old` substring with the `new` substring inside the string.
  • Case Methods:
    • `.capitalize()`: Capitalizes the first character of the string, and lowercases the rest.
    • `.title()`: Capitalizes the first letter of each word in the string.
    • `.upper()`: Converts all letters in the string to uppercase.
    • `.lower()`: Converts all letters in the string to lowercase.
    • `.swapcase()`: Changes uppercase letters to lowercase, and lowercase letters to uppercase.
